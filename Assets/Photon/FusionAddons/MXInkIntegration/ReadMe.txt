# MXInkIntegration

Sample of how to use Logitech MXInk with Fusion.
See https://logitech.github.io/mxink/UnityIntegration.html for their latest integration package.

## Dependencies

- XRShared > 2.0.5
- DataSyncHelper > 2.0.4
- LineDrawing > 2.0.0
